# Tim的读书笔记
我是一名游戏程序员，有一个技术博客：[Tim's Blog](http://wuzhiwei.net)。

这是我的读书笔记集群，笔记将全部用markdown编写。

技术类一般不包括编程语言学习书籍。

非技术类不涉及小说类。
